## Color Legend

- **Work Events**: Blue (#2196F3)
- **Personal Events**: Green (#4CAF50)
- **Birthday Events**: Deep Orange (#FF5722)
- **Reminder Events**: Orange (#FF9800)
- **Holiday Events**: Purple (#9C27B0)

## Interaction Flow

1. **App Launch**
   - Loads default view from SharedPreferences
   - Generates example events for 4 months
   - Displays calendar at current date

2. **Swipe Navigation**
   - Swipe left: Move forward
   - Swipe right: Move backward
   - Amount varies by view type (1 day, 3 days, 14 days, 1 month)

3. **View Switching**
   - Tap view button (2W, M, 3D, 1D)
   - Calendar recreates with new view type
   - Returns to current date/period

4. **Settings**
   - Tap settings button
   - Select default view
   - Changes persist across app restarts
   - Back button returns to calendar

## Screen Sizes

- **Optimized for**: 3840x2160 landscape
- **Also works on**: Any landscape Android tablet
- **Minimum**: 1280x720
- **Text scales**: Based on sp units
- **Touch targets**: 64dp minimum
